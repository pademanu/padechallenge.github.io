HTML elements position with CSS

Positions

atribute css ruleset description
Static default The default value (it does not need to be specified)
Relative position: relative; Allows you to position an element relative to its default static position on the web page.
Absolute position: absolute; The element will be positioned relative to its closest positioned parent element
Fixed position: fixed; Fix an element to a specific position on the page (regardless of user scrolling)
Sticky poisition: sticky; Keeps an element in the document flow as the user scrolls, but sticks to a specified position as the page is scrolled further

adicional info

Relative, Absolute, Fixed & Sticky need offset properties to work correctly

*offset properties: Top, Right, Bottom & Left*
